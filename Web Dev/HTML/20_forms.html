Forms

  • We can use forms to take data from user.
  • <form> tag is used to declare a form object.


  • For making a user input section , we use <input> tag.
  • There are a lot different types of <input> tags ,here in example only text and submit ones are shown.
  • We can change between different types of input tags using "type" attribute


  • We should make it a practice to make a <div> wrapper for every inputs of a form.

Form Attributes

  • 'method' [GET/POST] - backend stuff, by default GET
  • 'action' [specifies where the user inputs from form will end up stored, we can specify the file/webpage...etc name to it] - by default it stores in current webpage

'size' Attribute

- sets the size of the box [viewport]
- If size = '50' then it will make a box that can show 50 charecters at a time. - 'size' does not affect how much charecters that box can hold, it can hold way more (if set so) but will display only that many charecters how many are set in 'size' attribute
