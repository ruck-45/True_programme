Loading Image

- We load images using (img-tag), it is an empty HTML tags.
- it has following tags :
1. src [the source of image file]
2. height [set custon size in pixels]
3. width [set custon size in pixels]
4. alt [the description of image (shown when the image is not found, also web browser reads it in place of actual image)]
Example : here height = 720 px, width = 1080 px
