Table

  • (table-tag) is used to define a table object
  • (tr-tag) is used to define table rows
  • (td-tag) is used to define each individual data in that row
  • Table can include any data (text,image,hyperlink....etc)
  • (th-tag) is used to define table header
    • Becomes column headers if all declared in the first row.
    • Becomes row headers if defined one by one in each columns.
  • (caption-tag) is used to give table name
  • Header, Body , Footer
    • Header : The topmost row [usually the one containing column names]
    • Body : The rest of the table excluding header and footer
    • Footer : If the table is too long that reaching bottom row hides header, we use another row at the end identifying the column names for information

    • We can seggragate our table into header, body, footer(if any).
    • This wont make any changes in UI, but will help browser understand our webpage better and makes use of CSS easier.

    • (thead-tag) is used to define header
    • (tbody-tag) is used to define body
    • (tfooter-tag) is used to define footer

  • Attribute : border [give table a border of specified pixel]
