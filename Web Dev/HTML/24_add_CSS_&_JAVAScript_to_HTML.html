Adding CSS

  • Method 1 : [Using <style> tag]
    • We can directly write CSS code inside <style> tag which is inside <head> tag
    • Syntax (for this blue background):
      <style> body{ background-color: cornflowerblue; } </style>

  • Method 2 : [linking a CSS file using <link> tag]
    • We can instead link a css file in our directory using <link> tag
    • Syntax (for this blue background):
      <link rel="stylesheet" type="text/css" href="./CSS/style.css">

Adding JavaScript

  • Method 1 : [Using <script> tag]
    • We can directly write JS code inside <script> tag which is inside <head> tag
    • Syntax (for the alrt: 'WELCOME'): <script> alert('WELCOME'); </script>

  • Method 2 : [linking a JAVAScript file using <script> tag]
    • We can instead link a JAVAScript file in our directory using the same <script> tag
    • Syntax (for the alrt: 'WELCOME'):
      <script src="./JAVAScript/script.js"> </script>

<noscript> tag

  • we use <noscript> tag to display a message when the script cannot load in a browser, or the browser has disabled javascript.
  • It is also present in head-tag
  • Syntax : <noscript>Please enable javascript.</noscript>
