div & span


  • (div - tag) and (span - tag) are wrappers to other html tags.
  • They set all height, width, dimentions and specifications in which other tags will appear.

  • (div) is a block element.
  • (span) is an inline element.

  • Block and Inline element :
    • Block elements :
      • Block-Level elements take up the full width available to them once created (streaches out in both left and right direction as far as it can)
      • Block element starts on a new line when created.
