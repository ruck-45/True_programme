Favicon

  • Favicon is the icon that is displayed before the title in a new tab
  • Written using <favicon> tag.



  • Method 1 [using link - tag] :
    • Syntax :<link rel="icon" href="./images/favicon.png" type="image/png" sizes="">

      • cons :
    • now a days, websites are not just for desktops, people also open it on phones and tablets ..etc.. who have different aspect ratios.
    • To adjust our logo for those device we have to write differently.
    • So for that we use METHOD 2

  • Method 2 [using favicon generator] :
    • Just search favicon generator in google, select devices you want it to appear, it will generate the code.
    • Copy the code and paste it in <head> tag
    • Download the dataset they provide [different versions of your image]
    • put it in your root directory.
    • modify the code access address accordingly.
