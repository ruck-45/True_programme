Lists

There are 3 types of lists in html :
1. Ordered list
2. Unordered list
3. Description List

Note : Lists can be nested

Ordered List

- (ol-tag) used for ordered list. Represents collection of datas having a perticular order.
- (li-tag) is used to make each individual list element. - Attributes :
1. type [specifies which type of numbering, 1 - numerical , A/a - alphabetical(big/small), I/i - roman(big/small)]
2. start [specifies the starting value of that number , e.g we can start counting frpm 100, instead of 1]

Example :

  1. Coffee
  2. Chocolates
    1. Choco pie
    2. Hajmola
    3. Ecliers
    4. Center Fruit
  3. Biscuits
    1. Good Day
    2. Chaska Maska
  4. Salted Cashew
  5. Peanut Butter

Unordered List

- (ul-tag) used for unordered list. Represents collection of datas not having a perticular order.
- (li-tag) is used to make each individual list element.
- Attribute : style [used to specify the type of symbol used for list]

Example :

  • Coffee
  • Chocolates
    • Choco pie
    • Hajmola
    • Ecliers
    • Center Fruit
  • Biscuits
    • Good Day
    • Chaska Maska
  • Salted Cashew
  • Peanut Butter

Description List

- (dl-tag) used for Description list. Represents data having a title and a description.
- (dt-tag) is used to make each individual list element.
- (dd-tag) is used to give description about each individual list element.
- Attribute : style [used to specify the type of symbol used for list]

Example :

Coffee
: a hot black liquid
Milk
: a white liquid
