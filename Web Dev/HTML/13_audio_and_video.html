Audio

  • (audio-tag) is used to make an audio element
  • (controls) atttribute is used to setup the audio player and give control to users
  • (source-tag) is used to provide address of original audio file
  • We can provide multiple audio file types such that the browser can choose which file it supports and then play it.
  • We can add a text to notify users if the browser doesnot support any of the file we provided inside (audio-tag).

Video

  • (video-tag) is used to make an audio element
  • (controls) atttribute is used to setup the audio player and give control to users.
  • (autoplay) attribute can be used to take the controls away from user and directly autoplay in background.
  • (source-tag) is used to provide address of original video file.
  • We can provide multiple video file types such that the browser can choose which file it supports and then play it.
  • We can add a text to notify users if the browser doesnot support any of the file we provided inside (audio-tag).

  • (width and height) attributes are present to specify dimentions.
