Button
        - .btn : applies button functionalities to the element
        - can be used with button, a, input etc tags..
        - .btn-(color) : gives background color to button
        - .btn-outline-(color) : gives outline color to the button
        - .btn-lg and .btn-sm L control the size of the button
        - use : data-bs-toggle="button" property in button to give toggle functionality to the button (use .active class to start button in the active class)
      

        Button Group
        - Groups multiple buttons under a wrapper
        - .btn-group : class given to wrapper holds buttons
        - .btn-group-lg and .btn-group-sm L control the size of the button group
      

        - We can use it with checkboxes and radio buttons
        - the labels here become the buttons
      

        Button Group Vertical
        - .btn-group-vertical : gives the vertical variation
      

        Close Button
        - .btn-close : makes button a close button
        - give property  :  data-bs-theme="dark" to make the theme dark
