Progress
        - used to make progress bars

        - .progress : (Wrapper)
        - aria-valuemin="0" aria-valuemax="100" : (Wrapper) min max values
        - we can give height and width to determine the size of the bar
        
        - .progress-bar : the actual progress bar
        - we can give width to display the progress of the content
        - we can write content in it as progress lables
        - .bg-(color) : provides color to progress-bar
        - .progress-bar-striped : creates striped patten for progress bars
        - .progress-bar-animated : Animates progress-bars
    
        Progress bar stacked
        - .progress-stacked : (Wrapper above .progress), groups progress bars just like other grouping
