scrollspy
        - we can navigate to certain part of an html using scrollspy
        - usually is attached to some navigation bar/lists

        - Make your nav bar/ list 
        - make the content tab that each list corresponds to  (Wrap them in a wrapper) 
        
        - for each nav item (anchor tag) provide the targeted element ID in href attribute

        - In content Wrapper use following settings : 
            - data-bs-spy="scroll"
            - data-bs-target="ID of Navbar"
            - data-bs-smooth-scroll="true/false"
