Dropdown
        - .dropdown : (Wrapper of dropdown) , contains button and ul (for the list)
        - .dropdown-center :  make the list appear centered below the button

        - .dropup : (Wrapper of dropdown), list appears upwards
        - .dropup-center :  make the list appear centered above the button

        - .dropend : (Wrapper of dropdown), list appears right side
        - .dropstart : (Wrapper of dropdown), list appears left side
        
        
        - .dropdown-toggle : for the button/a responsible for the dropdown
        - data-bs-toggle="dropdown" :  property of the button/a 
        - data-bs-offset="10,20" : to change the offset of dropdown menu
        - data-bs-auto-close="true/inside/outside/false" : to tweek the autoclose feature of the dropdown

        - .dropdown-menu :  class on the ul that contains the list 
        - .dropdown-menu .dropdown-menu-(breakpoint)-start : left aligns menu
        - .dropdown-menu .dropdown-menu-(breakpoint)-end : right aligns menu


        - .dropdown-item :  on each list content  ( we can use button or a as a menu item)
        - .dropdown-header : for a header effect
        - .dropdown-divider : to create divider
        - We can place texts, forms , anything inside dropdown ...
