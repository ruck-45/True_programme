Placeholder
        - provides a loading placeholder until the element renders
        - always provide widthto placeholder (in form of col/w/custom width)

        - .placeholder : creates a placeholder of that perticluar element type, (use span to create placeholders inside the elements)
        - .placeholder .placeholder(lg/sm/xs) :  controls placeholder size
        - .placeholder-(glow/wave) : (parent), controls the animation of the placeholder  
        
        - .bg-color : to give background color to placeholders

        - Image support is not there (use svgs)
