.img-fluid
        - Used to make image responsive and scale to parents's width
        - It makes img max-width: 100%; and height: auto;, so height can go beyond parent's
    
      .img-thumbnail
      - .img-fluid + gives a rounded 1px border appearance to image
    

      .rounded-*
      - makes border of images rounded by specified unit
      - Works with any element
      - doesn't work with .img-thumbnail
rounded-5 

      .mx-auto
      - mx : margin in x axis
      - If we have single element , we can center it with mx-auto as it gives same margin to both sides of the element
      - It works with any element
mx-auto on col
