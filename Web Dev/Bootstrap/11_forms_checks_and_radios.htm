.form-check (wrapper)
        - Class that wraps checkboxes and radio buttons
        - For every wrapper (div) of checkboxes and radio buttions use  : .form-check
        - it is advised to have each radio button and check box inside a different wrapper to vertically stack them
    
        .form-check-input (important)
        - customizes check buttons and radio buttons
    
        .form-check-label
        - customizes check labels and radio lables
    


        .form-switch (wrapper)
        - Add .form-switch to the wrapper to convert a checkbox to a switch
    

        .form-check-inline (wrapper)
        - To avoid stacking of check boxes/radio buttons/switches
        - It shows them in same line
        - applied on the wrapper
        - use it along with .form-check
    

        .form-check-reverse (wrapper)
        - to put checkboxes, radios, and switches on the opposite side
        - applied on the wrapper
        - use it along with .form-check
