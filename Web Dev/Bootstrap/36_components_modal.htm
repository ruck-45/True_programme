Modal Window
        - .modal : (Wraper)
        - .modal .fade : to add fade animation to modal
        - data-bs-backdrop="static" : the modal will not close when clicking outside of it
        
        - .modal-dialog : (Dialogue box of the modal)
        - .modal-dialog modal-sm/lg/xl : controls size of the modal
        - .modal-dialog .modal-dialog-scrollable : To make scrollbar appear for modal if its too long
        - .modal-dialog .modal-dialog-centered : to vertically center modal dialogue box
        - .modal-dialogue .modal-fullscreen : always fullscreen modal 
        - .modal-dialogue .modal-fullscreen-(breakpoint)-down : fullscreen below the breakpoint
        
        - .modal-content : content of the modal
        
        - .modal-header : Header ( has a title and a close btn)
        - .modal-title : title of the header

        - .modal-body : Has body related content

        - .modal-footer : has footer content (usually byttons)
      

        Modal Button (controls the visibility of the modal)
        - data-bs-toggle="modal" : property specifies modal button
        - data-bs-target="#ID" : provide the modal id here
