Cards
        - A card is a flexible and extensible content container.
        - .card : (Wrapper)
        - use text alignment on card wrapper (text-start, text-center, text-end) to align contents inside card-body 
        - .text-bg-(color) : to give background color to card 


        - .card-header : header
        - .card-footer : footer 

        - .card-img-top : if image used at top
        - .card-img-bottom : if image used at bottom
        - .card-img : if image used as background 

        - .card-body : (wrapper for content) 
        - .card-title : title inside body 
        - .card-subtitle : subtitle inside body 
        - .card-text : text inside body 
        - .card-link : anchor tags inside body 

        - instead of card body use .card-img-overlay to use card-img as a background ( Note : content will overflow if they exceed image height) 

      
        Card Groups
        - Groups multiple cards under it
        - .card-group : (Wrapper)
