Collapse
        - animates height/width to have the collapse effect

        - data-bs-toggle="collapse" : property for button/a which facilitates the collapse
        - data-bs-target="#id" : defines the target of the button (used only for button)
        - if using anchor tag use href = '#id'

        - .collapse : class on div that is supposed to collapse
        - .collpase .collapse-horizontal : horizontal manner (fix a width of the content, otherwise it will make variable width accordin to text throughout the animation)
