Tooltips
        - simplified popovers
        - data-bs-toggle="tooltip" : to enable Tooltips
        - data-bs-title="message" : to give custom message in tooltips
        - data-bs-placement="top/bottom/left/right" : gives positioning to the tooltip popover

        - It can be used with anything

        - Include popper.js for support

        - Use below js for setup : 
            const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]')
            const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => new bootstrap.Tooltip(tooltipTriggerEl))
