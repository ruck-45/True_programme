Accordion (FAQ BOX)
        - Accordion has following classes in order : 
          
          1 .accordion (wrapper) : the whole accordion box
            2 .accordion-item : each of them represents a accordion box
              3 .accordion-header : the question wrapper
                4 .accordion-button .collapsed: The question part, use collapsed when using .collapse in .accordion-collapse below to show the button behavioyr as collapsed
                  5 Question (content)

              3 .accordion-collapse .collapse/.show: the Answer/Explanation wrapper, collapse closes the box by default, while show keeps it open by default
                4 accordion-body : the answer part
                  5 Answer (content)

        Note
         - If we are placing multiple independent accordions in a page, change
            - ID of .accordion
            - data-bs-target and aria-controls of .accordion-button
            - ID and data-bs-parent of .accordion-collapse
      

        Always open
         - Remove data-bs-parent attribute on each .accordion-collapse to make accordion items stay open when another item is opened.
