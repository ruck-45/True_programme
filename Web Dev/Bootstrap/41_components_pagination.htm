Pagination
        - .pagination : (wrapper), used in ul 
        - .pagination .pagination-(lg/sm) : size of pages
        - It uses flexbox, so use flex properties like (justify-content, align-items etc...)

        - .page-item : for each li , it can have icons, numbers, symbols etc..
        - .page-link :  for each anchor tag in li

        - .active : for active page-item
