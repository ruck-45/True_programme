Navbar
        - .navbar : (Wrapper) , it must include a container-fluid inside
        - .navbar .navbar-expand-(breakpoint) : (Wrapper) expands navbar on and above the mentioned breakpoint
        - use .bg-(color) :  to give color to navbar
        
        - .navbar-brand : for logo/Brand Name (can be p/a/div)
        
        - There is a button (navbar toggler) for smaller devices

        - .collapse .navbar-collapse :  used for responsiveness of navbar in smaller devices 

        - .navbar-nav :  used on (ul/ol/div) that holds all navbar items
        - .navbar-nav .navbar-nav-scroll: to allow scrollbars when nav is small in compacted from
        - .nav-item : holds all navbar items individually
        - .nav-link :  link for each navbar item
        - .navbar-text : for texts in navbar
    

        Navbar Placements
        - .navbar .fixed-top : fixed at the top of the viewport
        - .navbar .fixed-bottom : fixed at the bottom of the viewport
        - .navbar .sticky-top : sticks to the top of the viewport
        - .navbar .sticky-bottom : sticks to the bottom of the viewport
