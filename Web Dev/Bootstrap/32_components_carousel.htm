Carousel
        - It is slideshow for cycling through a series of content ( can be text, image , video etc..)
        
        - .carousel : (Wrapper) , marks the wrapper to contain carousel
        - .slide : default sliding animation of carousel
        - .slide .carousel-fade : add crossfade as the sliding animation
        - data-bs-ride="carousel" : use this property to enable autoplay
        - data-bs-touch="false" : property to disable touch swiping

        - .carousel-indicators : (indicator Wrapper), contains all indicators at the bottom of the carousel
        - data-bs-slide-to="slide-number(0-indexed)" : use this property to attach the indicator to a perticular slide
        
        - .carousel-inner : (content-Wrapper), contains all contents of the carousel
        - .carousel-item : (each content Wrapper)
        - data-bs-interval="10000" : use this property to set custom time for each item for autoplay
        
        - .carousel-caption : class for caption
        

        - .carousel-control-prev/ .carousel-control-next : classes for nav buttons
