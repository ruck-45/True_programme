col-(breakpoint)-*
        - It is used to specify specific layout for specific breakpoints
        - if we specify layout for a specific breakpoint: 
           
            ex : col-lg-3 , All the breakpoints equal or above it will receive same layout, while all breakpoints below will receive 100% width
                lg, xl, xxl -->  col-lg-3
                md, sm, extra-small  --> col (equal to col-12)

        - We can choose to not write "col-12" as it is similar to col (default version)

        X-Small       0 - 575 px              [.col-*]
        Small        576 - 767 px            [.col-sm-*]
        Medium       768 - 991 px            [.col-md-*]
        Large        992 - 1199 px           [.col-lg-*]
        X-Large      1200 - 1399 px          [.col-xl-*]
        XX-Large     >=1400 px               [.col-xxl-*]

        Auto space assign by system
        - If we provide grid values to some elements and not to others, it will divide the remaining space evenly among them
        - If no grid values are given it will divide them equally across all elements
        - It follows the grid hirarchy rule
