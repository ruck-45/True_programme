Alerts
        - Alerts in bootstrap doesn't come baked in, we have to use Js to implement their behaviour
        - Can look in documantation for its API and methods to use it
        - Here we will talk more about the styling of alerts
      
      Classes and roles
        - .alert : use on div that is the alert box
        - .alert-(color) : used to give color to alert box
        - role = 'alert' : use on alert div to specify its role
      
        Dismissing
          - .alert-dismissible : add to alert div to allow it to be alert-dismissible
          - .fade .show : add to alert div to trigger animation when it dismisses

          - The alert box shoud contain a "button" for closing
          - add .btn-close to the close button
          - give button attribute : data-bs-dismiss="alert"

          - It can cause foucs issues (if closed with keyboard), use Js to solve it (manually give focus to nearest element after tracking the dismissal)
