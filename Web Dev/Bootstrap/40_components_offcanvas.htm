Offcanvas
        - It is structurally similar to a modal
        - .offcanvas-(breakpoint) : (Wrapper)\
        - .offcanvas .offcanvas-(start/end/top/bottm) : controls the placement of the offcanvas
        - data-bs-scroll="true" : to enable body scrolling while on offcanvas
        - data-bs-backdrop="false" : to disable backdrop of offcanvas
        - data-bs-backdrop="static" : to make static backdrop (non interactive)

        - .offcanvas-header :  (contains offcanvas-title and cross-btn)

        - .offcanvas-title : header title 

        - .offcanvas-body :  (body of offcanvas, can have anything inside)
    

        Offcanvas Toggle Button
        - It can be a button or anchor tag
        - give the IF of the offcanvas in : href="#offcanvasExample" ( if anchor tag ) or data-bs-target="#offcanvasExample" ( if button )
        - use data-bs-toggle="offcanvas" attribute in both a/button
