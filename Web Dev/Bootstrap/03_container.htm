Grid-system : Container
        There are 3 types of containers in Bootstrap : 
            1. container
            2. container-fluid
            3. container-(breakpoint)
    
        Refrence Table
        

                        Extra-small     Small   Medium  Large   X-Large XX-Large
                            <576px      ≥576px  ≥768px  ≥992px  ≥1200px ≥1400px
        .container	    100%	540px	720px	960px	1140px	1320px
        .container-sm	    100%	540px	720px	960px	1140px	1320px
        .container-md	    100%	100%	720px	960px	1140px	1320px
        .container-lg	    100%	100%	100%	960px	1140px	1320px
        .container-xl	    100%	100%	100%	100%	1140px	1320px
        .container-xxl	    100%	100%	100%	100%	100%	1320px
        .container-fluid    100%	100%	100%	100%	100%	100%
    
        container
        - It leaves a side margin for all display ranges except extra small
        - Extra small gets full width
    

        container-fluid
        - It assigns 100% width for all display ranges
    

        container-(breakpoint)
        - It is a modified version of container
        - It applies container perperty for all display ranges equal or above the breakpoint mentioned
        - All lower break point get assigned 100% width

        E.g : container-lg : 
            - assigns side margin for : lg, xl and xxl
            - 100% width for extra small, sm and md
