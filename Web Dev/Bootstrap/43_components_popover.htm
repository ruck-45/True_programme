Popover
        - usually used with buttons
        - data-bs-toggle="popover" : property enabling popover
        - data-bs-title="Popover title" : property gives popover title
        - data-bs-content="popover content" : property gives popover content 
        - data-bs-placement="top/bottom/right/left" : property used for placement


        - include in your script and make sure pooer.js is included in bootstrap bundle: 
            const popoverTriggerList = document.querySelectorAll('[data-bs-toggle="popover"]');
            const popoverList = [...popoverTriggerList].map((popoverTriggerEl) => new bootstrap.Popover(popoverTriggerEl));
    
        Dismissable Popover
        - can only be done using anchor tags 
        - data-bs-trigger="focus" : use this property for dismissable feature 
        - tabindex="index" : is a must include
