Input Groups
        Input Groups have 3 components : 
        1. .input-group (class given on the Wraper that wraps all input groups together to be displayed as a single element)
        2. .input-group-text : (the content part of the input group, can be text, image, link, form/input anything)
        3. .form-control : (the form element)

        Input groups can have as manu input-group-text and form-elements as possible
      
        .input-group-(size)
        - controls the size of the input group
        - used along side of input group
        - We have following sizes available to us
            - .input-group : default
            - .input-group-sm : small
            - .input-group-lg : large
