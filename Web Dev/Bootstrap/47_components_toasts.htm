Toasts
        - behaviour similar to modals and offcanvas

        - Toast classes : 
            - .toast : (Wrapper)
            - .bg-text-(color) : give background color

            - .toast-header : header of toast (contains logo(if required), title, close button and aditional info)

            - .toast-body : has content of toast

        - use custom js to attach toasts to buttons
                const toastTrigger = document.getElementById('liveToastBtn')
                const toastLiveExample = document.getElementById('liveToast')

                if (toastTrigger) {
                const toastBootstrap = bootstrap.Toast.getOrCreateInstance(toastLiveExample)
                toastTrigger.addEventListener('click', () => {
                    toastBootstrap.show()
                })
                }


        - .toast-container : (wrapping above toast), used to group together multiple toasts, which will stack after one another after being triggered
