Nav
        - gives base for all kinds of navigation Components
        - .nav : (Wrapper), like ul/ol/div
        - .nav-item : nav items (li)
        - . nav-link : for anchor tags/buttons

        -  .nav uses flexbox so can utilize all flex classes here (like justify-content, align-items etc)
        - use .flex-column to make nav vertical

        - nav-fill : (wrapper), to fill up navigation space, but elements don't take equal width
        - nav-justified : (wrapper), to fill up navigation space, but elements take equal width

        Tab
        - gives base for all kinds of navigation Components
        - .nav .nav-tabs: (Wrapper), to generate a tab interface
        - .nav .nav-pills : (Wrapper), to generate a pill interface
        - .nav .nav-underline : (Wrapper), to generate a underline interface

        - .active marks the active tab

        - data-bs-toggle="tab" : property
        - data-bs-target="#ID" : provide the id of the target it corresponds to
        - aria-orientation="vertical" :  use it in (wrapper) to make tab vertical ( add a wrapper outside with flex display for it to work)
      
        Tab Content
        - Lets us move around the tab  ( separate from .nav wrapper)
        - .tab-content : (wrapper)
        - .tab-pane : for each tab we have in nav

        - .fade (on tab-pane): to add animation
