.bg-(color-code)
        - gives background color to element
        - We have following color codes :
bg-primary : blue
bg-secondary : gray
bg-success : green
bg-danger : red
bg-warning : yellow
bg-dark : dark gray
bg-light : light gray
bg-transparent : bg-transparent
bg-info : light blue
bg-white : white
bg-black : black
bg-body : white
bg-body-secondary : light gray
bg-body-tertiary : light light gray

        .bg-gradient
        - gives gradient to any color applied to the element
