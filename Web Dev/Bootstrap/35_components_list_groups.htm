List Group
        - It is just like Button Group
        - Works with ul/ol/ any div and repeated elements inside like (div/a/button/p/etc...)

        - .list-group : (Wrapper/parent element), used on outer div/ol/ul tags
        - .list-group-item : used on each item inside the parent (is available to all tags)
        - .list-group-item list-group-item-(color) : gives color to list groups ( add .list-group-item-action for a/buttons used as list-group-items)

        - .list-group .list-group-flush :  (Wrapper/parent element) to remove borders and rounded corners
        - .list-group .list-group-numbered :  (Wrapper/parent element) makes the list numbered
        - .list-group .list-group-horizontal : makes the group horizontal
