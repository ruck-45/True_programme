        .table-(bootstrap-color) (any element in table)
        - If used in a perticular cell, applies color to that cell
        - If applied on the table, it colors the whole table
        - These are the color varients available for table : 
            - table-primary : blue
            - table-secondary : gray
            - table-success : green
            - table-danger : pink
            - table-light : light pink
            - table-dark : black
            - table-warning : yellow
            - table-info : light-blue
