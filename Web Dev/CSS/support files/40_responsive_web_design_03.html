  • The Innitial width is set to be 70%, 30% beween Div - 1 and Div - 2.
  • Around 650px the Div - 2 box distorst its orientation to accomodate the large font size, so we changed the width to - 50%, 50% each.
  • We wanted to change the background-color for certain resolution, so we did that for 200px - 400px.


    • Media Query code:

    @media screen and (max-width: 650px){ #div_01{ width: 50%; } #div_02{ width: 50%; } } @media screen and (max-width: 400px) and (min-width: 200px){ #div_01{ background-color: crimson; } #div_02{ background-color: darkblue; } }



  • min-width = all width above specified width (at least)
  • max-width = all width below specified width (at most)
  • we can use "and" , "or" to specify the range of resolution we want.
