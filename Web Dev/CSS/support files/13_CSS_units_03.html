vh and vm

  • vh and vm (viewport height and viewport width) provides measure relative to the viewport-size.
  • viewport is the current size of opened screen
  • since we can change viewport accordin to our requirement, this unit allocation is dynamic.
  • the size will change if we change the size of the window. (try it on this site)
