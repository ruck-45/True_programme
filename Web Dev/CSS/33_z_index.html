z-index

  • Here is a setup:
  • if we wanted to bring the green one to front or the orange one to front then, we can do that by assigning the z-index.
  • z-index is the priority given in z axis (if screen is conidered x and y plane)
  • z-index has no limit on its value
  • The higher the z-index , the more hih up it is over others

    • Example :

    z-index of green = 3, orange = 2, black = 1


    z-index of green = 2, orange = 3, black = 1

  • z-index works on only those elements who are positioned. static elements will not respond to z-index.
