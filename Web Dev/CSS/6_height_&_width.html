Height & Width

  • We can define custom height and width of design elements (background/border... etc) here.
  • By default values are set to 'auto' (takes up only the amount of space required by container)
  • The values can be given in pixel (px) or Percentage (%).
  • The value given in pixel will be fixed to all devices.
  • The value given in Percentage(%) however considers the size of the container in which it is placed (div/span/body..etc) and allocates the specified percentage of height and width. It becomes flexible as the resolution of device changes.
