object-fit & object-position

  • we remember using background-size : cover / cotain etc... to fit backround image to our viewport dimentions.
  • Also we used background-position: top center / bottom left etc... to set the position of backround image.


  • we can use same operations on a video or an image using object-fit and object-position

    • Example of object-fit: (by default object-fit is "fill")

    width: 300px; height: 150px; object-fit: fill; (default, changes image dimentions to fit to container)


    width: 300px; height: 150px; object-fit: contain; (will show the whole image)


    width: 300px; height: 150px; object-fit: cover; (will cut the image according to aspect ratio)


    Example of object-position:

    width: 200px; height: 500px; object-fit: cover; object-position: right;


    width: 200px; height: 300px; object-fit: contain; object-position: top;
