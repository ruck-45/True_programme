Initial Vs Inherit

  • Every property has initial and inherit option as their values.
  • initial is the default value that is stated in CSS.
  • inherit applies the value given to its parent.

    • Example :

    This is div text.

    custom value : (color: red;) we provided custom color value
    link 1

    initial value : (color: initial;) default hyperlink color value is applied
    link 2

    inherit value : (color: inherit;) the color value of parent is applied
