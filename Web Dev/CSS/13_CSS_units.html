CSS Units

  • we use length in : width, height, margin, padding, border, font-size etc...
  • There are two types of units in CSS to provide length:
    1. Absolute Unit in CSS :
      1. cm (centimeter)
      2. mm (milimeter)
      3. in (inch)
      4. px (pixel)
      5. pt
      6. pc
    2. Realtive Unit in CSS :
      1. em
      2. rem
      3. vh
      4. vw
      5. % (percentage)

Absolute Units

  • Among all absolute units only px (pixel) is mostly used.
  • Comparision of all absolute units :
    1. cm (centimeter)
    2. mm (10 mm = 1 cm)
    3. in (inch)
    4. px (1 inch = 96 px)
    5. pt (1 inch = 72 pt)
    6. pc (1 pc = 12 pt)


    • Example :-
    width: 30cm;

Relative Units

  • All relative units :
    1. em - (relative to size of its direct parent)
    2. rem - (relative to root [html tag] size)
    3. vh - (1% of view port height)
    4. vw - (1% of view port width)
    5. % - (relative to parent)


    • Example :-
  • em
  • rem
  • vh & vw
  • % (percentage)
