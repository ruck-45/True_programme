Responsive Web Design

  • Earlier when mobile phones were not able to use internet, websites were only accessible only by PCs.
  • So websites were only developed for desktops.
  • But as more and more devices got access to internet, and website has to be displayed in multiple aspect ratios , thus there went a need to make the website responsive, such that they become readable in maximum devices possible.
  • At first companies started making separate website for smartphones starting domain with "m.website.com".
  • But Responsive website is much more economical way.
  • Some companies still dont have responsive wesite to promote their mobile apps and other products.

viewport meta tag

  • it is one of the fundamental methods of making website responsive.
  • syntax: < meta name="viewport" content="width=device-width, initial-scale=1.0" >
  • Here name="viewport" , suggests we are assingning some changes to the viewport of the device. Viewport is besically the visible screen of a device.
  • width=device-width , tells browser to not assume all devices as desktop and take the device width as actual width in consideration.
  • initial-scale=1.0 , restricts browser to zoom out thinking the width of device is of a desktop, and allows proper size implementation on other devices.

    • Example: [tap inspect after right click to compare different devices]
