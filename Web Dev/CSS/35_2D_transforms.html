Transforms (2D)

  • we can translate (move), rotate, scale or skew any element using "transform" property.
  • 'transform' property is only applicable on block / inline-block elements. It cannot be applied on inline elements.
  • To apply "transform" on inline elements we have to change their display property.

  • transform: translate(x_traslate , y_translate);
    • translate lets us move to specified position.
    • now we must be wondering what is its use, we are already able to define positions before.
    • We can use this to show different behaviours at different stages of an element.
    • Values here can be given in negetive to represent other axes.
    • transform: translateX() ; & transform: translateY(); can be used when we want to translate to only one perticular axis.

      • - Example : (We have applied translate to following anchor tag at hover stage, try hovering on it) [also we have to change display settings to inline-block]

      transform: translate(5px, 5px); [hover]
      Click Here

  • transform: rotate(degree-of-rotation);
    • rotate lets us turn the element wrt its center.
    • Values here can be given in negetive to rotate anticlockwise.

      • - Example : (We have applied rotate to following anchor tag at hover stage, try hovering on it) [also we have to change display settings to inline-block]

      transform: rotate(30deg); [hover]
      Click Here

  • transform: scale(horizontal-scale, vertical-scale);
    • scale lets us make the element big/small across both axis.
    • The "horizontal-scale" & "vertical-scale" provided are multiplied with the width and height respectively.
    • So to reduce size use fractional values and to increase size, use values above 1.
    • transform: scaleX() ; & transform: scaleY(); can be used when we want to scale to only one perticular axis.

      • - Example : (We have applied scale to following anchor tag at hover stage, try hovering on it) [also we have to change display settings to inline-block]

      transform: scale(0.8,2); [hover]
      Click Here

  • transform: skew(horizontal-skew [in degrees] , vertical-skew [in degrees]);
    • skew lets us make the element bend across both axis.
    • transform: skewX() ; & transform: skewY(); can be used when we want to bend to only one perticular axis.

      • - Example : (We have applied skew to following anchor tag) [also we have to change display settings to inline-block]

      transform: skew(15deg, 10deg);

  • transform: matrix( scaleX( ), skewY( ), skewX( ), scaleY( ), translateX( ), translateY( ) );
    • it is a shorthand used to apply scale, skew, and translate at the same time.
    • values should be entered as per the order that is specified above.
    • Donot write 'deg' or 'px' while giving value in matrix( ).

      • - Example :

      transform: matrix(0.8,5,5,0.8,10,10); [hover]

  • transform-origin: x-value y-value;
    • It allows us to set the origin from where transform properties are applied like(rotate/skew etc..).
    • values can be iven in words or percentage.
    • like :
      • top left (0% 0%)
      • bottom left (0% 100%)
      • top right (1000% 0%)
      • bottom right (100% 100%)
      • center center (50% 50%)

      • - Example :

      transform-origin: bottom right; transform: rotate(20deg);
