Box Shadow

  • It is used to provide shadow to container.
  • Syntax -> box-shadow: horizontal-shift vertical-shift bloom-radius spread-radius color ;
  • horizontal-shift & vertical-shift set the offset of shadow. Its value can be given in negetive to represent the offset in the opposite direction.
  • bloom-radius sets the blurryness of shadow.
  • spread-radius sets the spread of shadow.
