Giving two values in Border Properties

    - giving two values (separated by space) in border properties makes the first value applied to top and bottom part, and second value to right and left part.

    - Example :
  • border-style : dashed solid;
  • border-width : 5px 8px;
  • border-color : red green;
  • border-radius : 15px 0%;

Giving four values in Border Properties

    - giving four values (separated by space) in border properties makes the first value applied to top part, second value applied to right part, third value applied to bottom part and fourth value applied to left part.
    - 1st 2nd 3rd 4th = top right bottom left (clockwise)

    - Example :
  • border-style: solid dashed double dotted;
  • border-width: 8px 2px 5px 4px;
  • border-color: yellow blue violet red;

Giving value to a specific part in Border Properties

    - to give value to a specific part just write that part name in border property.

    - Example :
  • border-style: solid;
  • border-right-color: saddlebrown; (top/left/bottom/right for edges)
  • border-left-color: greenyellow;
  • border-top-right-radius: 16px; (top-right/top-left/bottom-right/bottom-left.. for vertices)
