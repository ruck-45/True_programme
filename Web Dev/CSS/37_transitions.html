Transitions

    - we use transitions to make changes smoothly within a time limit.

  • transition-property: (property-name-1), (property-name-2), (property-name-2)...;
    • specify the property which is set to change and we want to apply transition to it.
    • For multiple properties we can write them separated by comma.
    • As a shortcut we can write "all" to include all changeable properties at once.
    • If we decide to write "all", we cannot give custom duration to each property.


  • transition-duration: (time in s or ms);
    • transition-time is used to specify the duration over the transition will take effect.
    • for multiple transitions, we can specify different times using commas
    • the time can be specified in seconds(s) or miliseconds(ms)


  • transition-delay: (time in s or ms);
    • It specifies the delay after which the transition will happen
    • the time can be specified in seconds(s) or miliseconds(ms)


  • transition-timing-function
    • It specifies the way transition behaves.
    • Default value is linear, where the speed is same throught the time.
    • There are other varients as well like : ease-in(slow starting), ease-out(slow ending), ease-in-out(slow starting & ending but fast in middle).... etc


  • transition: (property) (time) (timing-function) (delay)
    • It is a shorthand for writing transition.
    • for multiple entries provide values for one entry space separated and then start the values of next entry comma separated.


    • Examples:

    • Hover over boxes to see effects.
    • the changed properties are applied to hover state & transition to base tag.


    • Without Transition


    • With Transition
      • [transition-property: width; transition-duration: 1s;]

    • transition-property: width,background-color ; transition-duration: 3s,1s ; Multiple property transition


    • transition-property: all; [width,background-color] transition-duration: 2s; "all" in transition-property


    • transition-property: all; [width,height,background-color] transition-duration: 2s; transition-delay: 2s; delay


    • transition-property: all; [width,background-color] transition-duration: 2s; transition-timing-function: ease-in-out; timing-function


    • transition: width 3s, background-color 1s ease-in-out; transition shorthand
