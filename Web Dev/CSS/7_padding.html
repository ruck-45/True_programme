Padding

  • It is the distance between the container and the actual content in it.
  • We can visualize the padding by giving a border to the container.
  • The distance between the the border and inner text from all its edges is the default padding given to it by the browser.

    • Visualization - (the distance between two borders is padding)

  • To customize padding space using padding property.
  • 'padding property' acts just like 'border property'.
  • Values can be given in pixel(px) or percentage(%).
  • If a single value is given , it is applied to all sides.
  • If two values are given it is applied to (top,bottom and right,left) respectively.
  • If four values are given it is applied in order of (top,right,bottom,left) order.
  • If we want to give value to a certain part we have to include its name in the syntax like : [ padding-top: (value) ;]

    • Example -

    - padding: 25px;
