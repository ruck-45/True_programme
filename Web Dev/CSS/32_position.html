Types Of Position

- There are 4 type of position:

  1. static
  2. relative
  3. absolute
  4. fixed
- There are four properties that help us define the position in webpage :
  1. top
  2. bottom
  3. left
  4. right
Note : the top, bottom, right and left properties can only be applied if the element is positioned, i.e, if the element has position set to : relative/asolute/fixed. They cant be applied to static.

position: static;

  • By default position is set to static.
  • It is the default position of elements predefined by CSS.

position: relative;

  • It allows us to move the element relative to its current position.
  • top, bottom, left, right can be used to provide the position.
  • Note : The preview of the element might move,but the position it initially held will still be occupied by it.

    • Example :

    - setup :


    - position: relative; top: 5px; left: 250px; [see how the original position of orange block is still unocuppied]

position: absolute;

  • It allows us to move the element relative to position of its nearest positioned parent.
  • When an element is positioned is absolute, it checkes its parent to be positioned (either relative/absolute/fixed).
  • If the parent is not found positioned then it looks at its parent's parent and the cycle goes on until it finds a positioned parent or reaches the "body" as the final parent.
  • top, bottom, left, right can be used to provide position.
  • Note : The element will no longer occupy its former space.

    • Example :

    - position: absolute; top: 1270px; left: 450px; [the div container was not positioned so the top and left are relative to 'body']


    - position: absolute; top: 5px; left: 250px; [here the div container is positioned]

position: fixed;

  • It fixes the element in the viewport and the element doesn't move even when scrolled up or down.
  • top, bottom, left, right can be used to provide position.
  • The position is fixed wrt. viewport.
  • Note : The element will no longer occupy its former space.

    • Example :

    - position: fixed; bottom: 10px; right: 10px;
  • The hovering orange cube at right bottem is the example created here.
  • The hovering help/chatbox in website have fixed positions.
