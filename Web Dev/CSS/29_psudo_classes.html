Psudo Classes

  • Psudo classes is really helpful if the website is dynamic (content is dynamic).
  • There are veriety of psudo classes in CSS.
  • Psudo classes are various states of a tag, like anchor-tag has states : hover, visited, active, link...
  • Syntax to use a psudo-class :
    • selector:psudo-class{ CSS-code; }

    Examples :

    first-child : first child
    last-child : last child
    nth-child(even) : for even pattern
    nth-child(odd) : for odd pattern
    nth-child(3n) : custom pattern
    nth-child(3n+1) : custom pattern
    nth-child(4) : directly select position
    nth-last-child(2) : position starting from last child

  • This psudo-class is for input tag.
  • when we click on input box it gets highlighted, it's the focus stage of input tag, and we can customize it.

    • Example:

    input:focus{ border: 5px dotted red; }

NOTE : if we want to select tags inside a certain tag then we can specify both selectors one after another separated by sapce to select it.

Example :

#div_01 p:nth-child(3){ color: blue; } [selects 3rd p tag inside the div tag]
