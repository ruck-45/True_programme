Transforms (3D)

  • transform: perspective( value in px ) ...
    • perspective is the measure of depth of the converging line in 3D
    • In 3d we get a cone like view where everything converges at a point, that depth is given by "perspective( )"


  • perspective-origin: x-coordinate(in %) y-coordinate(in %);
    • perspective-origin is the coordinate of the point where the perspective converges at.
    • It is just like "transform-origin"


  • transform: perspective( value in px ) translate3d(x-shift , y-shift , z-shift) [in px]
    • translate3d( ) lets us move element in 3D space.
    • Example :

      • transform: perspective(10px)
      translate3d(10px,0px,0px);
      x-axis

      transform: perspective(10px)
      translate3d(0px,10px,0px);
      y-axis

      transform: perspective(10px)
      translate3d(0px,0px,-10px);
      z-axis ;





  • transform: perspective( value in px ) rotate3d(x-shift , y-shift , z-shift , degrees-of-rotation) [shifts in px]
    • rotate3d( ) lets us rotate element in 3D space.
    • Example :

      • transform: perspective(100px)
      rotate3d(10, 0, 0, 20deg);
      x-axis

      transform: perspective(100px)
      rotate3d(0, 10, 0, 20deg);
      y-axis

      transform: perspective(100px)
      rotate3d(0, 0, 10, 20deg);
      z-axis ;
