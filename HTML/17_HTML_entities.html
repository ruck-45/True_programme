Html Entities

  • There are some special symbol reserved in html as keywords such that we cant use them freely as text.
    • Example: we cant write (<)table(>) without the parenthesis because < > is used to declare a tag & the moment we take off the parenthesis they make table a tag.
  • To use html entities as regular symbols we can do 2 things :
    1. By name:
      • syntax : &name;

        • Example: <table> or <table>

        [lt = less than]
        [gt = greater than]
    2. By number:
      • syntax : &#number;

        • Example: <table> or <table>

        [#60 = less than]
        [#62 = greater than]

  • We can look up names/numbers of html entities on the internet for use.

Non Breaking Space (nbsp)

  • Non Breaking Space is actually a html entity.
  • Html usually overrites all extra spaces and new lines in text.
  • we have a fix for new line using <br> tag .
  • The fix for extra spaces is Non Braking Space (nbsp)
  • Put (nbsp) in By name syntax of Html entities to use it.


    • Example :
  • This line has four spaces between every word but HTML has overridden it.
