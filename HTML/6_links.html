Links



- we use (a-tag) [anchor tag] to provide hyperlinks.
- attributes :
1. herf [the address to go to]
2. target [to open in a separate tab]
3. title [to provide information when hovered over the link]

Global Hyperlink

Can go to any html page.
Target attribute not used : Go to Google in this tab.
Target attribute used : Go to Google in new tab.
To know about text formatting, click here

Email Hyperlink

Can openup email software installed and compose an email (with pre set subject if we wish).
To ask any queries mail to : thoughtsfirstborn01@gmail.com

Local Hyperlink

Can jump to a specific paragraph in the same webpage.
But first we need to set up an id for each paragraph.
