Embed pdf

We can embed PDF in 3 ways :
  • (embed-tag) :

    Attributes : src [source] , type [document type], height & width [for dimention]

  • (iframe-tag) :

    Attributes : src [source] , height & width [for dimention]

  • (object-tag) :

    Attributes : data [source] , type [document type], height & width [for dimention]
