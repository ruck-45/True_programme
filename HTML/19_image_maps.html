Image Maps

Task : We have Following image (down) we want to make this image a hyperlink by parts.

Meaning :
If someone clicks on laptop pic, he will be directed to laptop page of amazon
If someone clicks on mouse pic, he will be directed to mouse page of amazon
If someone clicks on keyboard pic, he will be directed to keyboard page of amazon
If someone clicks on speaker pic, he will be directed to speaker page of amazon



  • We use image mapping in this case.
  • Method 1 :
    • create a <map> tag and give it a name.


    • inside the map tag, use <area> tags and give shape and coords [coordinates] attribute to define your area of intrest, in herf attribute provide the hyperlink to the targated website.
    • coordinates can be different for different shapes, e.g, for rectangle we just need to give two coordinates of its corners. For a circle we need to provide coordinate of its centre and provide its radius. And for a polygon we need to provide all necessery coordinates.


    • In <img> tag we use an attribute usemap [provide the map name in it] to use the map we just defined.
  • Method 2 :
    • Search image map generator in google, it will autometically generate code according to your requests
    • Just copy the code and paste in your html file.
    • Make sure to resize the image to correct resolution as in your html page and then upload to the image map editor.
