p-tag

- (p-tag) is used to write paragraph in a body. (this is written insde p tag)
- (p-tag) clears all formatting done inside the body and displays the palin text.
- We have to use other tags (e.g br-tag for new lines) for adding formatting inside pararaph.

- (lorem) is used to generate a large volume of random text.
- to generate a fixed number of text we need to specify alongside it (lorem50).
- the following is an edited (lorem) text but under p-tag it will appear without all spaces and indentatons and new lines we created for it.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Incidunt adipisci quia labore aperiam magni quam. Laborum hic dolore corporis inventore sit ad vel magnam minus quam. Quam, eos. Fugiat architecto labore cumque alias dolorum, quisquam quia! Quam tempora harum, sint omnis sed vero at ratione iusto architecto, quaerat aperiam natus!

- (pre-tag) will show the text as it is seen inside the editor, even with all indentations .(starting tabs and spaces)
- (pre-tag) is used when we need to give a lot of new line commands or spaces. (mostly while displaying codes) - in normal senarios (p-tag) can hold up preety well.
