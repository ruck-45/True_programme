<select> tag [makes dropdown form]

- <select> tag is used to create dropdown list in form.
- 'multiple' attribute lets user select multiple options at once.
- we can use 'size' attribute to set how many options can a person see at a time.

- Inside <select> tag we use <option> tag to make individual options.
- The name of the option that is displayed in the drop down list to the user is written between the opening and closing of (option tag).
- The value of the option can be set different to te face value shoen, to set the value, we use 'values' attribute.
- For example in form down below, one option is : <option value="5"> Botany </option> , here the user sees and selects Botany, but the value backend receives is 5.
- To make one of the option default value, use 'selected' attribute in thet option. (below statistics is choosen)

- <optgroup> is used to make groups of options in the same dropdown list, to give group names use 'label' attribute.
- the different subject groups below are made by <optgroup>.

<input type = "radio" > [makes circular buttons]

- Radio Buttens are primariely used where only one option is needed to be chosen out of multiple choices.


- To specify the options buttons represent, just write it in the right side.
- To group radio buttons such than only one of them is selectable, keep their 'name' attribute identical.
- To make a 'radio button' already chosen at start by default, use 'checked' attribute.
- Use 'value' attribute to pass required data to backend when that option is selected.

Datalist

- When we write anything in google's search bar, it gives us recommendations based on our letters.
- we can simulate this effect using datalist.
- we can create a datalist using <datalist> tag.
- inside datalist, we create our recommendations using <option> tag. [note donot close option tags here].
- Give the recommendation names as the 'value' attribute in <option> tag.
- then create an id of this datalist using 'id' attribute inside <datalist> tag.
- Now this datalist is ready to be used in any (text input -type). - To use this datalist , use "list" attribute inside the (input-tag) and provide the datalist id.
- Datalist will recommend all words that has the letter(s) just typed in.
