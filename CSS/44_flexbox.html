Flexbox

  • Flexbox is a relatively new addition to CSS.
  • It makes the positioning of container in HTML easier by mixing the functions of "inline-block", "float" & "position".
  • It is another display-type, and it makes all children of the element on which the display property is applied "flexible".

display: flex;

  • It makes the children of the element "flexible" and makes them available to "flexbox applications".
  • Majority of the flexbox properties is applied on the parent. Some are applicable on children for personal customization.
  • It is possible to achieve same results using "float", "positions". But the calculations involved will be very touh and complex.

Flexbox Properties For Parent-Box (to manipulate children)

  • flex-direction: (modes of direction) ;
    • Sets the way the boxes wil be aligned using various modes.
    • modes include :
      • row : default, left to right
      • row-reverse : right to left
      • column : top to bottom
      • column-reverse : bottom to top

    Examples :

    flex-direction: row; (default)

    three

  • flex-wrap: (modes of wrap) ;

  • justify-content: (different modes) ;
    • It lets us align the boxes in various ways in horizontal direction.
    • The default mode is : "flex-start"
    • modes include :
      • flex-start : default, left alignment
      • flex-end : right alignment
      • center : center alignment
      • space-around : gives equal amount of spaces in right and left of each box, the spaces distributed are the free spaces
      • space-between : equally divides free spaces in the region between the boxes, no space is provided at the extreme ends.

Flexbox Properties For Child-Box

  • order: (order of box) ;
    • lets us set the order through CSS irrespective of the order written in HTML.
    • Each child is iven an order number through which its order is defined.

    Examples :

    HTML order :
    one
    two
    three


    Custom order : three-(order: 1;), one-(order: 2;), two-(order: 3;)
    one
    two
    three


  • flex-grow: (ratio of growth) ;
    • flex-grow sets the ratio at which child-box will grow in size to fill in box if there are any empty spaces remaining.
    • The value given in flex-grow is relative to others (in ratio).

    Examples :

    normal setup :
    one
    two
    three


    flex-grow : one-(flex-grow: 1;), two-(flex-grow: 3;), three-(flex-grow: 1;) [box-"two" will grow three times the other boxes]
    one
    two
    three


  • flex-shrink: (ratio of shrink) ;
    • We know flexbox reduces individual width of child-boxes to accomodate all extra boxes in a single line.
    • flex-shrink sets the ratio at which child-box will shrink in size if the number of boxes increase.
    • The value given in flex-shrink is relative to others (in ratio).

    Examples :

    normal setup :
    one
    two
    three
    four
    five
    six
    seven
    eight
    nine
    ten
    eleven


    flex-shrink : one-(flex-shrink: 3;), two-(flex-shrink: 2;), three-(flex-shrink: 0;) [box-"one" will shrink three times, box-"two" will shrink two times, box-"three" will not shrink at all]
    one
    two
    three
    four
    five
    six
    seven
    eight
    nine
    ten
    eleven


  • align-self: (different modes) ;
    • "align-self" allows vertical alignment for each child-box individually.
    • The default mode is : "stretch"
    • modes include :
      • stretch : default, boxe stretches fully vertically to fill the height
      • flex-start : top allinment
      • flex-end : bottom allinment
      • center : center allinment

    Example :

    align-self: one-(align-self: flex-start;), two-(align-self: center;), three-(align-self: flex-end;)
    one
    two
    three


  • flex-basis: (value in px) ;
    • It sets the innitial width of a child-box.

    Example :

    two-(flex-basis: 300px;)
