Gradients

  • Gradient is the blending of diffrent
  • syntax : background-image: linear-gradient( mode, color1, color2, color3 );


    • - Example :

    background-image: linear-gradient(red, yellow, green);



    background-image: linear-gradient(to right ,red, yellow, green); [makes the gradient from left to right]



    background-image: linear-gradient(to left ,red, yellow, green); [makes the gradient from right to left]



    background-image: linear-gradient(to bottom right ,red, yellow, green); [makes the gradient go to bottom-right corner wise]



    background-image: linear-gradient(120deg ,red, yellow, green); [makes the gradient turn that much degree]



    background-image: linear-gradient(to bottom right ,red, yellow, green); [makes the gradient ellyptical pattern]



    background-image: radial-gradient(circle, red, yellow, green); [makes the ellyptical pattern of radial gradient circular]



    background-image: radial-gradient(circle ,red 20%, yellow 30%, green 50%); [sets the percentage of container space taken by each color]
