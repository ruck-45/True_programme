CSS_grid

  • Grid is a newer addition to CSS, newer than flexbox too.
  • The difference between flexbox and Grid-layout is : flexbox is good for 1D distribution, while grid is good at 2D distribution (merging rows and columns)



  • grid lines: imaginary lines that make up the table structure of the grid. Grid line is one more than the number of elements in that row / column.



Grid Display-property

  • display: grid;
    • It makes the display property of the element "grid", and allows its children to be available to "grid-properties".
    • It behaves as a block-element (div) taking up full width, makes each child a single row.

      • Example:

      one
      two
      three


  • display: inline-grid;
    • It makes the display property of the element "grid", and allows its children to be available to "grid-properties".
    • It behaves as a inline-element (span) taking up only required width, makes each child a single row.

      • Example:

      one
      two
      three

  • justify-content: (different modes) ;
    • It lets us align the Grid-layout (if extra spaces is available in width of parent) in various ways in horizontal direction.

      • end : right alignment
