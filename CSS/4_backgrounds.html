Properties under Background :

  • background-color : (color);
    • - applies backround color to the selected area
    background-color

  • background-image : url('(image path)');
    • - applies selected image as backround color to the selected area
    background-image

  • background-repeat : (no-repeat/repeat-x/repeat-y....);
    • - different ways to repeat or stop repeating
    - by default it is set to repeat
    background-repeat [no-repeat]
    background-repeat [repeat-x]
    background-repeat [repeat-y]

  • background-position : (x-pos y-pos);
    • - sets the position of image if decided to not repeat
    - the position can be given 3 ways :
    1. Using words like top,bottom,right,left,center etc... (Example - top right)
    2. Using Pixel to give exact position. (x-pixel y-pixel)
    3. Using Percentage to give exact position. (%-of-width %-of-height)
    top left
    bottom right
    right center
    center center
    50px 120px
    70% 40%

  • background-size : (x-size y-size);
    • - sets the size of image if decided to not repeat
    - the size can be given 3 ways :
    1. Using words like cover [will cut the sides of image to fully cover the background] ,contain [will show the full image, but unfilled background will remain blank] etc...
    2. Using Pixel to give exact size. (x-pixel y-pixel)
    3. Using Percentage to give exact size. (%-of-width %-of-height)
    cover
    contain
    800px 600px
    100% 100%

  • background-attachment : (fixed/scroll);
    • - sets wheather to fix the background or let it scroll with the content
    - by default it is set to scroll
    fixed

Shortcut to write background Properties in a single line

Syntax: - background : (background-color) (background-image url) (no-repeat) (fixed) (position [in %]) ;
- the order is mandatory, ommitting properties not required is acceptable.

Using Multiple Images

  • We can use multiple images by providing their ur separated by a comma.
  • For all other properties give that many sets of values separated by comma as many images being used.
