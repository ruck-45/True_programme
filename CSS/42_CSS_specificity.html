CSS specificity

If one element is styled by more than one rule, the which style will be applicable?

- the more specific rule will get more priority. (the rule havin more information about the element)

General Priority rules

  • "!important" will override everything.
  • "inline-CSS" can override everything except "!important".
  • "External-CSS" & "Internal-CSS" will work according to the "Specificity Rule"
  • if same rule is written two or more times in external/internal sheet, then the last rule will be applicable.

Specificity Rule

  • (#) id selector has highest value.
  • (*) universal selector has lowest value.


    • Point System to give value :

  • !important = 10,000 points
  • inline-CSS = 1000 points
  • id selector = 100 points
  • class selector, attribute selector, psudo-class selector = 10 points
  • element selector, psudo-element selector = 1 point
  • universal selector = 0 points

Example

<style> p{ background-color: blue; (1 point) } .bgcolor{ background-color: blue; (10 point) } div p:first-class{ background-color: blue; (1 point + 10 point = 11 points) } </style>

- The last one will be applied as its specificity is higher.
