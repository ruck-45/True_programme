  • We can provide margin space using 'margin property'.
  • 'margin property' value charecteristics is same as border and padding
  • we can give value in px or %
  • one value affects all sides, two and four value affect clock wise like in border.
  • we can also specify side name to apply margin to that side only like in border and padding.

Margin Collapse

  • Margin collapse is a feature that only affects the top and bottom part and not the sides.
  • In this, if a container has bottom margin and the bottom container has top margin, instead of both margin being added, the larger margin is just applied ignoring the smaller one.
- Example :

- margin-bottom: 20px (for upper para);
- margin-top: 50px (for lower para);
