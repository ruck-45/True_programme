Types of display property

  • Every element in HTML has a display property associated with it.
  • Four main Display Properties are :
    • block
    • inline
    • inline-block
    • none
  • By default, HTML elements are either : Block Element or Inline Element.
  • Inline-Block and None type are custom display properties used by us.

Charecteristics of various Display Properties

  • block :
    • It takes full width of viewport.
    • block elements start in new lines.
    • height and width can be customized using CSS.

  • inline :
    • It takes width that is required by the content.
    • inline elements donot start in new lines.
    • height and width cannot be customized using CSS.

  • inline-block : [it is besically inline element whose height and width is customizable]
    • It takes width that is required by the content.
    • inline elements donot start in new lines.
    • height and width can be customized using CSS.

'display' Property

  • 'display' property allows us to change the display of an HTML element.
  • we can change a block element to inline or inline-block and vice versa using 'display'

    • Example :

    - default block display. [height and width = 100px]

display: none; VS visibility: hidden;

  • display: none;
    • - it will completely remove the element and the space occupied by it from the webpage.

    Example:

  • visibility: hidden;
    • - it will just hide the element and the space occupied will remain in the webpage.
