Box Model

  • Box model is generally the layout of an element (container and its content) in HTML.
  • Box Model :-


  • Margin : space between two containers
  • Border : thickness of border given
  • Padding : space between the border and content
  • Content : the actual information inside the container (text/image/hyperlink/...etc)


    • Note : The height and width we set using CSS only applies to the content area.

    * To calculate the actual size of an element we must also have to add margin, border & padding :-

  • Total element width = content width + left padding + right padding + left border + right border + left margin + right margin
  • Total element height = content height + top padding + bottom padding + top border + bottom border + top margin + bottom margin
