calc() Function

  • It is used to give values in form of mathemetical expression.

    • Example :
  • Suppose we have following box -
  • we want to provide a space right side of 200px, and also when we change the viewport size the box can change, but the space in the right side should remain 200px always.
  • We can use calc() function here.
  • we will set - {width : calc(100% - 200px)}
  • Try changing the viewport , you will see the difference.

var() Function

  • We can create variables using var function.
  • Variables can be used to make big changes in website easily by just changing the values of the variables, instead of manually finding each value to change throughout the website.
  • Syntax :
:root{ --back-color: (color value); --text-color: (color value); --text-font: (font-family); etc... } #id{ background-color: var(--back-color); color: var(--text-color) etc.... }
  • If we need to cahnge background color, we can do it in :root and it will be applied to whole website.
