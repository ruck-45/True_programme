Print Style

  • We can customize the way our website looks when it is requested for a print. (ctrl + p)
  • We can exclude unnecessery menues, drop downs, header/footer, sidebars etc from appearing in the print version of our website using this method.
  • We use media queries to perform this task.
  • We can create specific CSS code for each different media, but we are going to focus on only print here.
  • syntax :

    • @media print { CSS code; }

  • "@media screen { }" can be used to apply the CSS code for vewing on screen (which by default is) . So we dont often use it.
