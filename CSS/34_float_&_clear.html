  • we did "float:right;" for both the images.
  • but we donot want them to be side by side, instead in a vertical manner.
  • For this reason we use clear.
  • clear can be use on both "float and non-float elements" but it will take action in the presence of a "float element" only.
  • like :
    • clear:left; - will not allow the selected element to exist on the left side of a floated element.
    • clear:right; - will not allow the selected element to exist on the right side of a floated element.
    • clear:both; - will not allow the selected element to exist on the left or the right side of a floated element.
  • we used "clear:right;" on the flower image to bring it down.
