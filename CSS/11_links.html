States of a hyper-link

  • There are 4 stages of a hyper-link
    1. link : unvisited stage. By default it is blue color with a underline.
    2. visited : when the site has been visited once ever in your browser, it goes into visited state. by default, in this state Hyper-link remains violet color with an underline
    3. hover : when curser is hovered on the hyperlink. By default hover state is defined with no style.
    4. active : when the hyper-link is clicked, it enters active state. By default, for the brief moment it is clicked, it turns into red color and retains its underline.


  • We can modify the look of our hyperlink using css.
  • The way to access each state of hyperlink is :
    • #id_name:state_name { CSS code }
    • .class_name:state_name { CSS code }

    • Example - (NOTE : keep the order mentioned below, do not write one state over other except the mentioned order.)
    #id_01:link { ..... }
    #id_01:visited { ..... }
    #id_01:hover { ..... }
    #id_01:active { ..... }

  • We can remove the default underline of links by setting {text-decoration : none;}
