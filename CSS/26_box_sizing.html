Box Sizing

  • By default the height and the width we provide is applicable to the content, not the container.
  • The margin, border, padding we provide are added over the height and width provided, thus it can make the container bigger than the specified height and width.
    • Example :

    box 1 : height: 200px; width: 600px;

  • box-sizing deals with this exact senario.
  • It allows us to apply the height and width to the container.
  • by default the value is - "content-box"
    • Example :
