"!important" property

  • "!important" property is used to set priorities.
  • it is used to override the priorities of 'inline' over 'id' over 'class' over 'elemental' CSS.
  • it is not recommended to use it in CSS.
  • it is used to quickly fix a CSS document if it is difficult to find the root css code.

    • But after writing "!impotant " after a property in any low tier CSS definition, it overrides the priority.


      • NOTE : "!important" only works on a single line of code, not entire block of code.

    • To demonstrate that we have included font-color: from elemental, font-size: from class, font-family: from id.


      • NOTE : if there are presence of multiple "!important" on same element of same property, then the priority rule from before kicks in.
