How To Use Google Fonts

  • go to google fonts.
  • select all fonts nad their varients you want to include.
  • copy the link code and paste in head tag of HTML file to include the font files.
  • Use font-family code given in google site to specify the font to be applied, in css code.


    • - Example :
    font-family: 'Licorice', cursive;

Font Fallback

  • It is advisable to uve multiple fonts in font family to make use of Font Fallback.
  • like : font-family : name1, name2, name3... ;
  • It is useful when the browser is uneble to use one font style because of some error(file not found/ file curropt), it will switch to immidiate next font mentioned.
